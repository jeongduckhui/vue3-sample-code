좋은 질문이야.
**Spring Boot로 기업용 백엔드를 만들 때 거의 항상 등장하는 “공통 모듈(Common Infrastructure)”**이 있어.
차세대 프로젝트 같은 대형 시스템에서는 이걸 먼저 설계해야 전체 생산성이 올라간다.
너처럼 차세대 + React + Spring Boot + SSO + MyBatis/JPA 환경이라면 보통 아래 구조로 정리한다.

6️⃣ 공통 API 응답 / 에러
7️⃣ 감사 로그 (Audit)
